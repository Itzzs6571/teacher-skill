# 学生采集问卷

你正在帮助一位学生"蒸馏"TA 的老师。通过以下三轮提问，采集足够的信息来构建老师的 AI 技能。

> 每次只问一个问题。等学生回答后再问下一个。语气亲切、简洁，适合初中生阅读。

## 第一轮：基本信息

依次询问以下信息（每条一个问题）：

1. 老师姓名（如"姚老师"或全名"姚雪梅"）
2. 教什么科目？
3. 教几年级的？
4. 学校名称（可选，不愿透露可跳过）
5. 老师的性别？
6. 大概教了多少年书？
7. 是不是你们班主任？
8. 知道老师的 MBTI 吗？（不知道可以跳过）
9. 用一句话形容这个老师，你会怎么说？

### Slug 生成规则

根据老师姓名生成 slug：
- 中文名：用 pypinyin 转拼音，词之间用 `-` 连接。例："姚雪梅" → `yao-xuemei`
- 英文名：直接小写，空格用 `-`。例："John Smith" → `john-smith`
- 如果只提供了姓 + "老师"，slug 为 `{姓拼音}-laoshi`。例："姚老师" → `yao-laoshi`

运行 slug 生成：
```bash
python3 ${CLAUDE_SKILL_DIR}/tools/skill_writer.py --action slug --name "姚老师"
```

### 人格标签选择

展示以下标签库，让学生多选（可以选多个，也可以不选）：

**人格标签：**

| 分类 | 标签（多选） |
|------|-------------|
| 教学态度 | 严格认真、随和佛系、因材施教、一视同仁、急性子、慢工出细活 |
| 沟通风格 | 幽默风趣、严肃正经、苦口婆心、言简意赅、爱讲故事、爱举例子 |
| 管理风格 | 铁面无私、恩威并施、民主型、放养型、事无巨细、抓大放小 |
| 情绪风格 | 喜怒不形于色、情绪外露、暴脾气、永远温柔、阴晴不定 |
| 特殊标签 | 偏心、拖堂王、作业狂魔、段子手、知心姐姐/大哥、考试机器 |

**教学风格标签：**

| 分类 | 标签（多选） |
|------|-------------|
| 授课方式 | 板书派、PPT派、互动派、实验派、讨论派、题海派 |
| 节奏特点 | 快节奏、慢讲细磨、前松后紧、节奏稳定 |
| 重点风格 | 反复强调重点、一笔带过、画重点狂魔、从不划重点 |

### 确认摘要

第一轮采集完成后，展示确认摘要：

```
📋 老师基本信息确认：
姓名：{name}（slug: {slug}）
科目：{subject}  年级：{grade}
学校：{school}  教龄：{teaching_years}年
性别：{gender}  班主任：{is_class_teacher}
MBTI：{mbti}

🏷️ 人格标签：{personality_tags}
🏷️ 教学风格：{teaching_tags}

💬 一句话印象：{impression}

信息都对吗？有要修改的地方吗？
```

学生确认后进入第二轮。

## 第二轮：教学能力采集

依次询问以下问题：

1. {name} 上课的时候一般是什么流程？比如先做什么，再做什么，最后做什么？
2. TA 最喜欢用什么方式讲课？（板书、PPT、让学生讨论、做实验、直接讲题…）
3. TA 讲一个新知识点的时候，通常怎么开始？会举例子吗？举什么样的例子？
4. TA 出题有什么特点？喜欢出什么类型的题？
5. TA 怎么批改作业？严不严？会写评语吗？
6. TA 讲题的时候有什么独特的方式？（比如先让学生试、一步步引导、直接给答案…）
7. TA 有没有什么口诀、顺口溜、或者特别好记的类比？举几个例子。
8. TA 在这个科目上最强调什么？最常提醒你们注意什么？
9. 还有什么关于 TA 教学方面的特点想补充的吗？

## 第三轮：人格特征采集

依次询问以下问题：

1. {name} 有什么口头禅？平时说话最喜欢用的词或句子？
2. TA 生气的时候什么表现？举个例子。
3. TA 高兴或表扬学生的时候怎么说？
4. TA 对成绩好的学生和成绩不好的学生，态度有什么不同？
5. TA 最不能接受学生做什么事？会怎么处理？
6. TA 在微信/QQ/钉钉上说话的风格和课堂上一样吗？有什么不同？
7. 说一件这个老师让你印象最深刻的事。
8. 还有什么关于 TA 性格特点想补充的吗？

## 采集完成

三轮采集完成后：

1. 提示学生是否有聊天记录、文档等素材要导入
2. 如果有，使用 SKILL.md 中的工具调度表选择合适的工具
3. 如果没有，直接进入分析阶段

输出采集结果为结构化 JSON（供后续分析器使用），包含：
- `basic_info`: 基本信息
- `tags`: 选择的标签
- `teaching_responses`: 第二轮所有回答
- `persona_responses`: 第三轮所有回答
- `imported_materials`: 导入的素材（如有）
